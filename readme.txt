●使用許諾

・J-POSTALは、フリー・ソフトウェアです。個人使用、業務使用のどちらでも自由に
　使用してください。


●J-POSTALの概要

　『郵便番号－住所検索の子ウィンドウ』のためのHTMLとJavaScriptです。
　
　☆J-POSTALは、あなたの住所入力フォームに『郵便番号－住所検索の子ウィンドウ』の
　　機能を追加します。
　
　☆データベースやCGIは不要です。
　
　☆設置作業の詳細は、MANUAL/ を参照してください。
